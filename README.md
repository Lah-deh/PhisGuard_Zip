PhisGuard - Phishing Detection Browser Extension
Overview
PhisGuard is a browser extension designed to help users identify and avoid phishing websites. The extension analyzes visited URLs and alerts users when a potentially malicious or phishing site is detected.
This project focuses on frontend logic, browser extension structure, and user-facing security feedback.

Key Features
Detects potentially phishing or unsafe websites
Provides real-time alerts to users
Lightweight and easy to install
Runs directly in the browser as an extension
User-friendly warning interface

Technology Stack
JavaScript
HTML
CSS
Chrome Extension APIs

How to Install (Manual Installation)

Download the project ZIP file and extract it
Open Google Chrome
Navigate to:
chrome://extensions/
Enable Developer mode (top right)
Click Load unpacked
Select the extracted project folder
The PhisGuard extension will be added to your browser

How It Works

The extension monitors the current webpage URL
URLs are checked against phishing detection logic
When a suspicious site is detected, the user is notified
Users receive clear visual warnings to avoid interaction
